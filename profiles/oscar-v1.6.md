Application profile for the OSCAR 1.6 serialization format. Relative to OSCAR
1.5, the current corpus introduces the newer typed encodings for `ZZRingElem`,
`QQFieldElem`, and phylogenetic trees, while matrix groups still use the older
`MatrixGroup` root type.
