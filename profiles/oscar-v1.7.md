Application profile for the OSCAR 1.7 serialization format. Relative to OSCAR
1.6, the main documented change in the current corpus is the matrix-group
rename from `MatrixGroup` to `MatGroup`; otherwise this profile matches the
OSCAR-specific encodings documented here.
